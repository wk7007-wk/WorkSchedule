# WorkSchedule 코드맵 (슬림 재작성 0417)
# 3파일 분리구조: app.js 374줄 + index.html 40줄 + style.css 122줄 = 536줄
# code-guard: 수정 전 이 맵으로 영향 범위 확인

## 파일 구조
docs/app.js      — 전체 JS (단일 IIFE)
docs/index.html  — HTML (모달 5개 포함)
docs/style.css   — CSS (다크테마 #1A1A30)

## app.js 섹션 (줄번호 → 역할)

### config (1~11)
FB, FW           Firebase URL 상수
DSH=3            하루 시작시각 — 게이지/정렬/타임바 전부 연동
DE               기본 직원 5명 (emp1~emp9)
HOL              공휴일 단일맵 {'yyyy-MM-dd':'이름'} (2026~2027)
RC/RL            역할 색상/라벨

### store (13)
S                전역 상태 단일 객체 {tab,date,emp,sc,wsc,fix,dof,cf,sst,att,sse*,loaded,sec}
  → S.emp = 직원, S.sc = 당일스케줄, S.fix = 고정스케줄, S.dof = 휴무, S.att = 근태

### util (16~33)
pad,dk,toast,openM,closeM,sameD,getMon  기본유틸
isH,hNm,isWE   공휴일/주말 판정
fEmp            이름→empId 검색
esc             XSS 이스케이프 (innerHTML 삽입용)
isDayOff,gSt    휴무/상태 조회
tm12,tmDS,cH,tPct,pTM  시간 계산

### api (35~37)
fbG(url)         GET  — .json suffix 자동
fbP(url,data)    PUT  — READONLY 체크 포함. data=null → 삭제 대체
  ※ fbPatch, fbDelete 없음 (PUT만 사용)

### SSE (49~65)
connectSSE()     직원 SSE + 스케줄 SSE 연결
conSS(gen)       스케줄 SSE (날짜별, 세대관리)
  → S.gen으로 세대 관리, 재연결 3초 backoff

### 고정스케줄 (38~47)
gFix(name,date)  = getFixedScheduleForDate — dayTimes 오버라이드 지원
  → S.fix[name].dayTimes.{dow}.{start,end,role} 우선 적용
  → type:'fixed' + off:[] 또는 type:'weekly' + days:[] 분기
gFixC(name)      현재 날짜 기준 gFix 호출

### 데이터 로딩 (67~84)
loadData()       초기 7개 병렬 fetch → store 세팅 → genDO → autoFix → renderAll
loadWk()         주간 7일 스케줄+상태 fetch → S.wsc 세팅

### 자동배정 (86~101)
autoFix(dk)      빈 스케줄에 고정값 자동 입력 + 자동확정
genDO()          56일간 자동휴무 생성 (고정스케줄 기반)

### 분류 (103~118)
catE(dk)         직원을 working/off/empty로 분류 + 주간통계
gRange(w)        타임바 게이지 범위 계산 (출퇴근+실근태 기반)

### 근태 (120~133)
attRow(eid,sh)   실제 출퇴근 오버레이 HTML (시간차+출처배지)
ASC/ASL          출처별 색상/라벨맵

### 렌더 (134~264)
renderAll()      RAF 디바운스 → doR()
rBrief()         브리핑 패널 (4칸: 출근/총시간/인건비/휴무 + 공휴일배지)
rTimebar()       타임바 뷰 (수평바 + 현재시각선 + 근태오버레이)
rList()          리스트 뷰 (미니바 + 주간시간 라벨)
renderWeek()     주간 개요 (휴무 칩)
renderDS()       날짜 스트립 (7열 그리드 + 확정색상)
rTab()           현재 탭에 따라 rTimebar/rList 분기
  공통 헬퍼: progBar, stBtn, emptyRow, offRow, sectWrap

### shift 모달 (271~307)
openSh(eid)      모달 열기 (칩선택+게이지+프리셋+역할)
bTS()            시간 select 옵션 생성 + 게이지 틱
setupGauge()     터치 드래그 시간 선택
shiftSave        저장 → fbP schedules + 자동확정
shiftDel         삭제 → fbP(url, null)
shiftFixed       고정값 저장 → fbP fixed_schedules
shiftDayoff      휴무지정/해제

### 직원 관리 (308~318)
rEL()            직원 목록 렌더
oEE(id)          직원 편집 모달 (이름/전화/색상/역할/시급)
empEditSave      저장 → fbP employees

### 휴무 관리 (319~335)
rDL()            휴무 목록 렌더
pBulk(text)      일괄 휴무 텍스트 파싱 (요일/범위/ISO/슬래시)
doAddBtn         단건 추가
doBulkBtn        일괄 등록

### 내비게이션 (336~353)
onDC()           날짜 변경 → SSE 재연결 + 4개 병렬 fetch + autoFix
openDP()         30일 날짜 선택기
showFlash()      날짜 전환 플래시

### 확정/상태 (266~270)
sSt(d,e,st)      개별 확정 토글
cfAll()          전체확정
togOff(eid)      휴무 토글 (해제 시 고정값 복원)
cfOff(eid)       휴무 확정

### 공유/기타 (354~372)
shareBtn         텍스트 공유 (이원규 제외)
urlBtn           URL 복사
collapsible      주간 개요 접기
visibilitychange SSE+loadData 재호출 (reload 금지)
init             S.date=new Date → updD → bTS → setupDel → loadData → connectSSE

## 의존성 그래프 (수정 시 영향)
gFix ← autoFix, gFixC ← rBrief
S.fix ← gFix, shiftFixed
S.sc ← loadData, SSE, autoFix, shiftSave/Del/Dayoff, togOff
S.emp ← loadData, SSE, empEditSave
catE ← rTimebar, rList (모든 뷰 렌더)
renderAll ← SSE put/patch, loadData, 상태변경 전부
